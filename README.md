<h2>项目整体结构</h2>
<ul>
  <li>vue-cli</li>
  <li>vue-router</li>
  <li>webpack</li>
  <li>node</li>
  <li>express</li>
</ul>
<h3>安装</h3>
<p>前提是需要本地电脑安装npm和node<p>
<h3>打开</h3>
<p>demo文件中打开cmd然后输入npm run dev</p>
<p>demo文件中的server文件打开cmd然后输入node index</p>
